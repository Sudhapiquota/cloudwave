CUSTOM CONSOLE APPLICATIONS

CloudWave has partnered with customers to integrate their on-prem solutions with Smartsheet via custom console applications. The application is hosted on the customer’s server and is easily maintained with no continued maintenance expenses. An example would be a healthcare client with an on-prem database that copies data from their EMR/EHR to Smartsheet for reporting to groups that need limited access. Smartsheet is HIPAA compliant, and measures can be taken in the solutioning process to ensure limited-to-no access to detailed information. CloudWave provides error logging and notifications, technical documentation, and other documentation as needed to ensure the continued success of the solution after hand-off.

BRIDGE - BI-DIRECTIONAL INTEGRATION WITH OTHER CLOUD SERVICES

CloudWave, as a trusted Smartsheet Platinum Partner, is working with various clients across multiple industries to integrate their Smartsheet instances with their other cloud platforms. Data is fed from identified fields in their ERPs, ITSMs/ITOMs, CRMs, cloud databases and more to Smartsheet and data is fed back as appropriate.

Using Smartsheet in this capacity allows companies to:

  • Leverage their data in a user friendly environment
  • Have all their data in one place
  • Easily create and automate reporting
  • Enhance visibility and collaboration
  • Easy maintenance in the no-code Bridge environment

BRIDGE - AUTOMATING REPEATING PROCESSES

Bridge is designed to automate repeated processes. CloudWave can capture business needs and create workflows to eliminate redundant tasks consuming your team's time. Examples of tasks that can be eliminated:

  • Updating dropdown values
  • Pulling tracking information from UPS or FedEx
  • Copy and pasting of data from other systems

CONTROL CENTER - PMO AND BEYOND

Control Center is a premium application that can automate the approval and rollout of new projects. CloudWave’s Smartsheet team can help capture requirements, advise on best practices and ways to gain efficiencies, and create a customized solution. Smartsheet’s new card view in reporting also allows for improved Agile project management within Smartsheet. In addition to Control Center’s ability to automate PMO activities, Smartsheet’s tools allow for great inventory management. Inventory management tools combined with Control Center capabilities allow for a robust inventory solution. Control Center can also handle archiving projects, which is ideal for companies with high volumes of projects.

WORKAPPS AS A PATIENT PORTAL

CloudWave has assisted a Lab customer in leveraging Smartsheet’s Document Builder, WorkApps, and Dynamic View to create a Portal:

  • In a sheet with lab results, Document Builder can be used to convert a result into a PDF report, which is attached to that row
  • Each row specifies the email associated with the clinic or specific person that needs to be able to view the result
  • The Dynamic View uses the sheet or report with the results and the current user functionality on the email, so the user logged in will only be able to see their rows
  • The Dynamic View can be embedded in the WorkApp, which is user friendly, easily accessible, and easy to navigate on both a computer and mobile device
